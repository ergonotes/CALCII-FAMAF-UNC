#### **Definición:** Una función $f$ de $n$ variables es...
una regla que asigna a cada $n$-tupla $\bar{x} \doteq (x_1, \dots, x_n)$ un único número real $f(\bar{x}) = f(x_1, \dots, x_n)$.
El dominio de $f$ es el subconjunto $\text{Dom}(f)$ de $\mathbb{R}^n$ dado por $$\text{Dom}(f) = \{\bar{x} \in \mathbb{R}^n : f(\bar{x}) \text{ está bien definida}\}$$El rango o imagen de $f$ es el subconjunto $\text{Im}(f)$ de $\mathbb{R}$ dado por
$$\text{Im}(f) = \{y \in \mathbb{R} : \exists \ \bar{x} \in \text{Dom}(f) \text{ con } y = f(\bar{x})\}$$
El gráfico de $f$ es el subconjunto $G(f)$ de $\mathbb{R}^{n+1}$ dado por
$$G(f) = \{(x_1, \dots, x_n, x_{n+1}) \in \mathbb{R}^{n+1} : (x_1, \dots, x_n) \in \text{Dom}(f) \text{ y } x_{n+1} = f(x_1, \dots, x_n)\}$$
$$= \{(\bar{x}, f(\bar{x})) \in \mathbb{R}^{n+1} : \bar{x} \in \text{Dom}(f)\}$$
****
#### **Ejemplos y más...**
**Ejemplo 1.** Determine el dominio de $f$ si
$$f(x, y, z) = \ln(z - y) + xy \operatorname{sen} z$$
**Solución.** La expresión para $f(x, y, z)$ está definida mientras $z - y > 0$, así que el dominio de $f$ es
$$D = \{(x, y, z) \in \mathbb{R}^3 \mid z > y\}$$
Este es un **semiespacio** que consta de todos los puntos que se ubican arriba del plano $z = y$. $\blacksquare$
****
Una función $f$ de tres variables es difícil de visualizar porque su gráfica tendría que representarse en un espacio tetradimensional. Sin embargo, se hará una idea de $f$ examinando sus **superficies de nivel**, las cuales son las superficies con ecuaciones $f(x, y, z) = k$, donde $k$ es una constante. Si el punto $(x, y, z)$ se mueve a lo largo de una superficie de nivel, el valor de $f(x, y, z)$ se mantiene fijo.
****
**Ejemplo 2.** Determine las superficies de nivel de la función
$$f(x, y, z) = x^2 + y^2 + z^2$$

**Solución.** Las superficies de nivel son $x^2 + y^2 + z^2 = k$, donde $k \ge 0$. Estas forman una familia de esferas concéntricas con radio $\sqrt{k}$. (Véase la figura) Así, cuando $(x, y, z)$ varía en cualquier esfera con centro $O$, el valor de $f(x, y, z)$ se mantiene fijo. $\blacksquare$
